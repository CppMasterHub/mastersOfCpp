Calling Function

Call by Value

In Call by Value, the actual values of the arguments are passed to the function. Any modifications made to the parameters inside the function do not affect the original values outside the function.

Example : Call by value

        
#include <iostream >

    // Function prototype
    void incrementValue(int x);
    
    int main() {
        int num = 5;
    
        // Function call by value
        incrementValue(num);
    
        // The original value remains unchanged
        std::cout << "Original value: " << num << std::endl;
    
        return 0;
    }
    
    // Function definition
    void incrementValue(int x) {
        x++;
        std::cout << "Incremented value inside function: " << x << std::endl;
    }

Screen Display :

    
        Incremented value inside function: 6
        Original value: 5

Call by reference

In Call by Reference, the memory address of the actual arguments is passed to the function. Modifications made to the parameters inside the function directly affect the original values outside the function.

Example : Call by reference

  
#include <iostream >

// Function prototype
void incrementValueByReference(int &x);

int main() {
    int num = 5;

    // Function call by reference
    incrementValueByReference(num);

    // The original value is modified
    std::cout << "Modified value: " << num << std::endl;

    return 0;
}

// Function definition
void incrementValueByReference(int &x) {
    x++;
    std::cout << "Incremented value inside function: " << x << std::endl;
}

Screen Display :


    Incremented value inside function: 6
    Modified value: 6

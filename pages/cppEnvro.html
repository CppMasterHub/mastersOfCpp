C++ Programming Environment

All C++ system generally consist three parts:

  1. The Environment
  2. The language
  3. C++ Standerd library

C++ programs typically go through six phones to be executed. These are: edit , preprocessor, compille , link , load , and execute.

Editor A program is created in the editor and stored on disk. Save it with .cpp extension.
Preprocessor The preprocessor program processes the code, performing tasks like text replacement and including other library files with the file to be compiled.
Compiler The compiler creates object code and stores it on disk.
Linker The linker links the object code with the libraries, creates a.out, and stores it on disk.
Loader The loader puts the program in memory.
CPU The CPU takes each instruction and executes it, possibly storing new data values as the program executes.

General Formatting Guidelines in C++:

  1. C++ doe distinguish between uppercase and lowercase lattes (i.e.,it's case sensitive).
  2. Every C++ program Statement must end in a semi-colon.
  3. Curly braces { and } are used group a set of expression together.
  4. Comments should be added to your code to clarify it (using either // short comments or /* ..... */ longer comments).

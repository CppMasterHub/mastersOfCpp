Structure Of C++ Programming

Here is a sample C++ program to illustrate its structure : hello.cpp

Example : Sample C++ program

        
/*HELLO.CPP : First C++ program */
#include <iostream>

int main() {
std::cout << "Hello, World!" << std::endl;
return 0;
}

Parts of C++ Program

  1. Comments -a type of program documentation
    2. // indicates that the remainder of the line is a comment
    /* comments can also look like this for multiple-lines */
  2. A preprocessor directive
    3. #include <iostream.h>

    Tells the pre-processor to include in the program the contents of the I/O stream header file called iostream.h.This allows us to use standard stream input and output objects like cout (displays to the screen).

  3. Main function header
    4. int main()

    Every C++ program has at least one function,called main. And there is ONLY ONE main. Program execution begins with first statement in main.

  4. Braces
    5. { }
    { brackets denote the body of the function }
  5. Statement terminator
    6. ;

    Every C++ statement must end with a semicolon.

  6. Stream insertion and exertion operator
    7. << and >>

    Expression to the right of the operator is insertes (sent) to the cout object (the diplasy screen.

    Exertion is totally opposite for it.

  7. Newline escape in an sequence
    8. \n

    The backslash is an on escape character.

  8. Exiting from the function
    9. return 0;

    In this case control over execution is transferred back to the operating system.

Variables

A variable is named memory location whose value can change during run time.Since a variable holds a data item,it has a type.

Rules For Naming Variables are as follows

  1. Variable name can be a combination of alphabets,digits & underscore.

    2. E.g. int a_123;

  2. First character of a variable name must be an alphabet.

    3. E.g. int a2,ghj;

  3. Variable name cannot include any other special characters such( ? , * , # , ; ) or a blank space.

    4. E.g. int a*2; //Not legal

  4. A keyword cannot be used as a variable name.

    5. E.g. float int; //Not legal

  5. In variable name,uppercase & lowercase are distinct.

    6. E.g. int a,A; //a & A are two different variables

Declarations Of Variables

Declarations inform the complier that it will need to set aside space in memory to hold an object of a particular type (class) with a particular name.A variable declaration tells the compliers

  1. Name Of Variable
  2. Data-Type i.e. which type of data will be stored by the variable.
A declaration statement for variables has the general form:
data-type variable-name;
For Example:
int total; //declares total
A variable must be declared before it can be used.

Initialization Of Variables

The syntax or rule for assigning initial value to thr variables during declaration is called initialization.
data-type variable-name = initialial-value;
For Example:
float pi = 3.14159 ;

Dynamic initialization of variables

A nice feature of C++ is that variable declaration and initialization may be freely intermixed and contained with any other statement; the only requirement is that a variable must be declared prior to its use. For very large programs this is a plus, because it allows variables to be declared close to where they will be used. The previous example can be re-written as :

                
    #include <iostream>
    using namespace std;

    int main() {
        float grade1 = 85.5,grade2 = 97.0;
        float total = grade1 + grade2;
        float average = total / 2.0f;       //divide the total by 2.0
        cout << "The average grade is " << average << endl;
        return 0;
    }

Screen Display :

        
            The average grade is 91.25

Reference Variables

Reference is an alias or another name or synonym for a variable having same value as its parent i.e. point to same memory location.

Example : Sample C++ program

        
/*HELLO.CPP : First C++ program */
#include <iostream>
using namespace std;

int main() {
    int a;
    int &r = a;
    cout << "Enter a ? " ;
    cin >> a;
    cout << "\nValue of a : " << a;
    cout << "\nValue of r : " << r;
    r = r + 10;     // if r=10 ,r=10+10=20
    cout << "\nNow value of a : " << a;
    cout << "\nNow value of r : " << r;
    return 0;
}

Screen Display :

        
            Enter a ? 14

            Value of a : 14
            Value of r : 14
            Now value of a : 24
            Now value of r : 24
